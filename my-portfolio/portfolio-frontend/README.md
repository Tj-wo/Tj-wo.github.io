# Portfolio Frontend

Quick start for the frontend (Vite + React):

Development:

```powershell
cd d:\Tj-wo.github.io\my-portfolio\portfolio-frontend
npm install
npm run dev
```

Build:

```powershell
npm run build
```

Notes:
- The Vite `root` is set to `src/` in `vite.config.js`.
- Styles are in `src/style.css` and imported from `src/main.jsx`.
